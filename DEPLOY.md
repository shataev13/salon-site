# Деплой на reg.ru (статика + живые цены через PHP)

Хостинг reg.ru — общий (PHP/Apache), без Node.js. Поэтому сайт собирается в
**статический экспорт** (`output: "export"`) — готовые файлы, которые кладутся
в `/www/shatistudio.ru`. Цены при этом остаются **живыми**: их подтягивает
небольшой PHP-прокси `prices.php` из Google Таблицы прямо в браузере.

## Сборка

```bash
npm ci
npm run build      # результат — папка out/ (уже включает public/prices.php)
```

Содержимое `out/` — это всё, что нужно положить в корень сайта
`/www/shatistudio.ru`. Загрузить можно архивом и распаковать на месте.

## Как работают цены

- Страницы услуг собираются с ценами, заложенными в коде (`lib/pricing.ts`) —
  показываются мгновенно и работают всегда.
- В браузере `components/LivePricing.tsx` запрашивает `/prices.php`, который
  читает CSV из Google Таблицы (с кэшем ~5 мин), и обновляет цены на лету.
- Разбор CSV — общий модуль `lib/price-build.ts` (та же логика, что при сборке).

Чтобы живые цены работали, таблица должна быть открыта на чтение по ссылке
(«Поделиться» → «Доступ по ссылке» → «Читатель»). ID/лист таблицы заданы в
`public/prices.php` и в `lib/sheet.ts`. Если прокси недоступен — остаются цены,
заложенные при сборке.

## Домен

`shatistudio.ru` должен указывать на сервер reg.ru (A-запись → `37.140.192.187`).
Почтовые записи (MX, mail/pop/smtp, TXT) не трогать.
