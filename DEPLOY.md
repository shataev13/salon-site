# Деплой на Vercel

Проект на Next.js — деплоится на Vercel «из коробки», без правок кода.

## 1. Подключить репозиторий

1. Зайти на <https://vercel.com> и войти через **GitHub**.
2. **Add New… → Project** → импортировать репозиторий `shataev13/salon-site`.
3. Framework Preset определится сам — **Next.js**. Build Command, Output и
   Install — оставить по умолчанию. Нажать **Deploy**.

## 2. Какая ветка деплоится

Vercel по умолчанию публикует ветку **`main`** (production), остальные ветки —
как preview-сборки по push.

Сейчас вся работа в ветке `claude/relaxed-bell-bmei7i`. Варианты:

- **Слить ветку в `main`** (через Pull Request на GitHub или локально), и Vercel
  будет собирать `main`. ← рекомендуется.
- Либо в **Project → Settings → Git → Production Branch** указать
  `claude/relaxed-bell-bmei7i`.

## 3. Переменные окружения (необязательно)

Всё работает со значениями по умолчанию (онлайн-запись и ID Google-таблицы
зашиты в коде). При желании их можно переопределить в **Settings →
Environment Variables**:

| Переменная                  | Зачем                                            |
| --------------------------- | ------------------------------------------------ |
| `NEXT_PUBLIC_BOOKING_URL`   | Ссылка на онлайн-запись YClients                 |
| `GOOGLE_SHEET_ID`           | ID Google-таблицы с прайсом                       |
| `GOOGLE_SHEET_GID`          | gid листа в таблице                               |
| `GOOGLE_SHEET_REVALIDATE`   | Период обновления прайса, сек (по умолчанию 300)  |

> Чтобы цены подтягивались из таблицы вживую — таблица должна быть открыта на
> чтение: «Поделиться → Доступ по ссылке → Читатель». Иначе показываются цены
> из кода (`lib/pricing.ts`).

## 4. Свой домен (reg.ru)

1. **Project → Settings → Domains** → добавить ваш домен.
2. Vercel покажет, какие DNS-записи прописать (обычно `A` на адрес Vercel
   и/или `CNAME`).
3. В личном кабинете reg.ru открыть управление DNS домена и добавить эти записи.
4. Дождаться обновления DNS — Vercel сам выпустит SSL.

## 5. Дальнейшие изменения

Правки → коммит → **`git push`** → Vercel автоматически пересоберёт и выкатит.
Отдельно «заходить на сервер» не нужно — деплой управляется через git.
