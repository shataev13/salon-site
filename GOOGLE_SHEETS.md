# Управление прайсом и услугами через Google Таблицу

Сайт может брать **услуги, их названия, категории и цены** из Google Таблицы.
Меняете таблицу — сайт обновляется (по умолчанию раз в ~5 минут), без правок кода
и пересборки. Если таблица не настроена — используются данные из кода.

## 1. Структура таблицы

Создайте таблицу с **двумя вкладками**.

### Вкладка «Услуги»

Определяет карточки/страницы услуг и их порядок.

| slug   | Название           | Описание                    |
| ------ | ------------------ | --------------------------- |
| hair   | Парикмахерский зал | Стрижки и укладки           |
| nails  | Ногтевой сервис    | Маникюр, педикюр, покрытие  |
| massage| Массаж             | Расслабление и уход за телом|
| spa    | СПА                | Обёртывания и программы      |

- **slug** — латиницей, без пробелов. Это адрес страницы: `/services/<slug>`.
  У существующих услуг slug лучше не менять (поломает ссылки). Чтобы переименовать
  услугу — меняйте только «Название».
- **Название** — как услуга показывается на сайте.
- **Описание** — короткий текст (в выпадающем меню и на странице услуги).
- Порядок строк = порядок услуг на сайте. Новая услуга = новая строка.

### Вкладка «Прайс»

Цены, привязанные к услугам.

| slug | Группа          | Название                | Цена     | Примечание            |
| ---- | --------------- | ----------------------- | -------- | --------------------- |
| hair | Мужская стрижка | Мужская стрижка         | 3 000 ₽  |                       |
| hair | Мужская стрижка | Спортивная стрижка      | 2 000 ₽  |                       |
| hair | Женская стрижка | Короткие волосы         | 4 000 ₽  |                       |
| nails| Маникюр         | Маникюр                 | 2 000 ₽  |                       |
| spa  | VIP             | «Тело и лицо»           | 13 400 ₽ | Обёртывание + уход…   |

- **slug** — к какой услуге относится строка (как во вкладке «Услуги»).
- **Группа** — подкатегория (показывается аккордеоном). Можно оставить пустой —
  тогда позиции идут без подзаголовка.
- **Название** / **Цена** — позиция и цена как есть (`3 000 ₽`, `по запросу` и т.п.).
- **Примечание** — необязательный текст под позицией.
- Если у услуги нет ни одной строки в «Прайс» — на странице покажется
  «Актуальную стоимость уточняйте у администратора».

> Названия колонок не чувствительны к регистру. Допустимы и английские варианты:
> `name`/`title`, `description`, `group`, `price`, `note`.

## 2. Подключение

**Вариант A — по ID (рекомендуется):**

1. В таблице: **Поделиться → Доступ по ссылке → «Читатель»**.
2. Скопируйте ID из адреса: `docs.google.com/spreadsheets/d/`**`ЭТОТ_ID`**`/edit`.
3. Добавьте в `.env.local` (локально) или в переменные окружения хостинга:
   ```
   GOOGLE_SHEET_ID=ЭТОТ_ID
   ```

**Вариант B — публикация в CSV:** «Файл → Поделиться → Опубликовать в интернете»,
выберите каждую вкладку как CSV и пропишите ссылки:
```
GOOGLE_SHEET_SERVICES_URL=https://.../pub?gid=...&output=csv
GOOGLE_SHEET_PRICES_URL=https://.../pub?gid=...&output=csv
```

Шаблон переменных — в `.env.example`. После изменения `.env.local` перезапустите
`npm run dev`. На проде изменения подхватываются автоматически
(интервал — `GOOGLE_SHEET_REVALIDATE`, по умолчанию 300 сек).
