# Прайс из Google Таблицы

Цены на сайте берутся из **Google Таблицы салона** (экспорт каталога
Яндекс.Бизнес — один лист). Меняете таблицу — сайт обновляется сам, по
умолчанию раз в ~5 минут, без правок кода и пересборки. Если таблица
недоступна (закрыта или нет сети) — сайт показывает цены из кода
(`lib/pricing.ts`), поэтому всегда продолжает работать.

## 1. Что нужно сделать один раз

Открыть таблицу на чтение:

**Поделиться → Доступ по ссылке → «Читатель»**

Этого достаточно — ID таблицы и нужный лист уже прописаны в коде
(`lib/sheet.ts`). После этого сайт начнёт подтягивать цены автоматически.

## 2. Структура листа

Лист — это экспорт из Яндекс.Бизнеса. Важны 4 колонки (регистр заголовков
не важен; подходят и английские варианты `category`/`name`/`price`/`description`):

| Категория          | Название         | Цена | Описание                         |
| ------------------ | ---------------- | ---- | -------------------------------- |
| Стрижки и укладки  | Мужская стрижка  | 3000 | Мытьё, стрижка и укладка.        |
| Ногтевой сервис    | Маникюр          | 2000 | Аппаратный маникюр…              |
| Обёртывания THALION| Бу-Марин         | 8500 | Грязевое морское обёртывание.    |

- **Цена** — число (`3000`); на сайте форматируется как `3 000 ₽`. Нечисловое
  значение (например `по запросу`) выводится как есть.
- **Описание** — полный текст; показывается под позицией. Если в группе один
  и тот же текст повторяется у всех позиций — он скрывается (чтобы не дублировать).
- **Категория** — определяет, в какой из 4 разделов сайта попадёт позиция (см. ниже).

## 3. Как категории попадают в разделы

На сайте 4 раздела. Категории таблицы распределены по ним в `lib/sheet.ts`
(константа `SECTION_SOURCE`):

- **Парикмахерский зал** — «Стрижки и укладки», «Окрашивание волос»,
  «Уходы и лечение волос (Lebel)».
- **Ногтевой сервис** — «Ногтевой сервис».
- **Косметология** — «Уходы для лица», «Пилинги», «Аппаратная косметология
  (Biogenie)», «Массаж лица», «Брови и ресницы», «Перманентный макияж»,
  «Макияж», «Депиляция воском (LYCON)», «Депиляция сахаром (шугаринг)».
- **Массаж / СПА** — «Массаж тела», «Талассо-массаж и коррекция тела (THALION)»,
  «Обёртывания THALION», «Метод AROSHA (коррекция фигуры)».

Каждая категория показывается на странице раздела отдельной группой (аккордеоном),
позиции — в порядке строк таблицы.

> Менять цены, названия, описания и состав услуг можно прямо в таблице.
> А вот **новая категория** (которой нет в списке выше) или **новый раздел**
> требуют небольшой правки `SECTION_SOURCE` в `lib/sheet.ts` — напишите, добавлю.

## 4. Другая таблица / лист (необязательно)

Если нужно указать другую таблицу — задайте переменные окружения
(шаблон — в `.env.example`):

```
GOOGLE_SHEET_ID=<ID из адреса .../spreadsheets/d/ЭТОТ_ID/edit>
GOOGLE_SHEET_GID=<gid листа из адреса ...#gid=ЭТОТ_GID>
# либо прямая CSV-ссылка:
GOOGLE_SHEET_PRICES_URL=https://.../export?format=csv&gid=...
# период перечитывания, сек (по умолчанию 300):
GOOGLE_SHEET_REVALIDATE=300
```

Локально переменные кладут в `.env.local` и перезапускают `npm run dev`.
На проде — в переменные окружения хостинга; изменения в таблице подхватываются
автоматически (интервал — `GOOGLE_SHEET_REVALIDATE`).
